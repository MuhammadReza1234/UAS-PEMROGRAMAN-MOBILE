## Isi Konten
### Activity Hello 
Activity Hello menampilkan kalimat dengan menggunakan string yang ada pada halaman kosong.

<img src= https://github.com/rniarzz/UAS-PEMROGRAMAN-MOBILE/assets/115542704/51878c45-c842-449c-b5f2-ccdb98f38877 width="250px" height="500px">

---

### Activity Count 
Activity Count menampilkan sebuah program untuk menghitung bilangan fibonacci, untuk menjalankan program ini terlebih dahulu masukkan jumlah max angka fibonacci, kemudian klik tombol `count` maka program akan menghitung angka fibonacci dimulai dari angka 0. Selanjutnya tombol `back` berfungsi untuk me-reset hitungan kembali ke angka 0

https://github.com/rniarzz/UAS-PEMROGRAMAN-MOBILE/assets/115542704/a76eb209-610a-441e-823f-10966d5d42e7

### Activity Message
<p align='justify' Di dalam Activity Message ini terdapat 2 activity yaitu activity satu & activity dua, jadi kedua activity ini saling berhubungan. Pada activity satu jika kita inputkan sebuah kata lalu klik `send` maka program akan mengirim pesan yang sudah di ketik sebelumnya dan menampilkannya pada activity dua.</
